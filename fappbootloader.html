Name scheme + instructions for use

Name scheme

Fmos-LEV1= FlameOs Locked EXE compatable version 1

Fmos-LEV2= FlameOs Locked EXE compatable version 2

Fmos-UEV2= FlameOs Unlocked EXE compatable version 2

Fmos-LBV3= FlameOs Locked BAT compatable version 3

Fmos-UBV3= FlameOs Unlocked BAT compatable version 3

Instructions for use

Fill out "filname.txt", "filwe.txt", "verinfo.inf" Before use. Place the previously named batch file (the file that was stated in "filname.txt" and "filwe.txt"). And then go to your FlameOs's app folder and create a new folder with the **same name** as your app.

If you have the unlocked version please do all the steps above and run mafsp.bat *Before changing some files* ("error.txt", "location.txt").
